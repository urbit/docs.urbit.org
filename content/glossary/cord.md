# Cord

A **cord** is an [atom](atom.md) [aura](aura.md) for a little-endian UTF-8 string in [Hoon](hoon.md). Its aura is `@t`, and its syntax is any string of text surrounded by single quotes like `'Foo bar baz!'`.
